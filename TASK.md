## 4-1. CDアイコン統一削除【優先度: 高】

**現状の課題**: `opticaldisc` アイコンが以下に残存している。ランキング画面では削除済みだが、他画面で不統一。

| ファイル | 箇所 |
|---|---|
| `HomeView.swift` | `HomeRankingRow` の楽曲タイトル横 |
| `TrackDetailView.swift` | タイトル横 |
| `LibraryView.swift` | `TrackRowView` の楽曲タイトル横 |

**提案**: ライブラリ・楽曲詳細・ホームからも削除。ただし「統計 > 音源の内訳」のアイコンとしての使用（CDのシンボル）は維持する。

### 4-5. 初回起動時の empty state 改善【優先度: 高】

**現状**: 初回起動直後は月別・年別レポートが "データはまだありません" の空状態のみ。ユーザーは何をすればデータが増えるか不明。  
**提案**: baseline / early 状態専用の説明画面を追加。

```
[スナップショット取得済み]
あなたのライブラリを記録しました。
次回以降の同期でリスニング履歴が蓄積されます。
今後 X 回の同期で詳細分析が始まります。
```
